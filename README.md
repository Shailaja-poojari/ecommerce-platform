# E-Commerce Microservices Platform

A scalable backend e-commerce system developed using Java, Spring Boot, and Microservices Architecture.

## Features

* User Management
* Product Catalog Management
* Order Processing
* Authentication and Authorization
* API Gateway
* Service Discovery
* Database Integration

## Technology Stack

* Java 17
* Spring Boot
* Spring Cloud
* PostgreSQL
* Docker
* Kafka
* Maven

## Architecture

The application follows a microservices architecture where each business capability is implemented as an independent service.

Core Services:

* User Service
* Product Service
* Order Service
* Payment Service
* Notification Service

## Learning Outcomes

* REST API Development
* Microservices Architecture
* Service Communication
* Database Design
* Containerization with Docker
* Backend System Design
